Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito utilizando a tag b (não semântica)

Já nesta tenho um termo em destaque usando a tag strong (semântica)

Itálico / Ênfase

Nesta tag temos um termo em itálico usando a tag <p> (não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Tambem posso utilizar a tag <mark> para um texto marcado

Texto grande e pequeno

Com a tag <big> podemos fazer um texto grande porém esta tag está obsoleta, ou seja, não é aconselhável usar-la
E com a tag <small> eu posso fazer um texto pequeno

Texto deletado

Eu posso também marcar um texto como excluído para indicar que ele pode ser lido, mas não considerado. Para isso eu posso utilizar a tag <del> (semântica)
Ou a tag <strike> o que não é aconselhavel (não semântica)

Texto inserido

Posso marcar um texto como inserido para dar enfase e indicar que ele foi inserido depois, para isso utilizamos a tag <ins> (semântica)
Ou a tag <u> (não semântica)

Texto sobrescrito e subescrito

Para escrever uma equação ou uma formúla química adequadamente podemos fazer uso das tags <sub> e <sup>.
Com a tag <sub> podemos escrever textos subescritos como H2O
E com a tag <sup> textos sobrescritos como x26+23x+12=0
